# Changemappers Open Data Repository

The Changemappers open data repository provides datasets, frameworks, and structured knowledge emerging from regenerative action research and community co-creation. The goal is to make insights, patterns, and tools openly accessible to accelerate collective impact and avoid duplicated effort across changemaker initiatives.

[https://changemappers.org/opendata/](https://changemappers.org/opendata/)
